"# Gestionnaire de commandes - Restaurant" 
 Le concept de ce projet est tout d'abord de travailler mes connaissances en php objet et de les améliorer, notamment avec l'implentation de twig. 
 Cette application représentera une caisse avec les plats les commandes ou encore les équipes du restaurant 

 Nous aurons dans cette application : 

 - Une page de login
 - Une page pour la gestion du menu du restaurant (plats, prix, ect..
 - Une page pour la gestion des commandes (ajout, historique, paiement..)
 - Une page pour la gestion du personnel (rôle, salaire, commandes réalisées...)

   CE PROJET EST EN COURS DE DEVELOPPEMENT
